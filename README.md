# OverTheWire-Bandit-WarGame
The repo is about bandit wargame of OverTheWire.org. This helps in learning of Linux basic commands. It is just a searching or finding the password from given file system from bandit server. In each level difficulty will be increased. 
This Repo will contains my solutions and technical notes for this game.

## My Goals
To master all the basic linux commads and practice technical documentation, and I completeing "2-3 Levels daily".
